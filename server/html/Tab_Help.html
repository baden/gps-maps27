Команды управления через SMS

КомандаОписание
configОбновить конфигурацию
gps offВыключить только GPS-модуль
gps onВключить только GPS-модуль
rebootПерезагрузить устройство
power offОтключить устройство
fwupdateОбновить внутреннее программное обеспечение
gsm rebootПерезапустить GSM-модуль
iparam параметр значениеУстановить значение параметра
iparam gsm.server 1Переключиться на сервер Пилигрим
iparam gsm.server 0Переключиться на сервер Google
iparam gsm.apn 0Настроить точку доступа для оператора Киевстар-предоплаченый.
iparam gsm.apn 1Настроить точку доступа для оператора Киевстар-контракт.
iparam gsm.apn 7Настроить точку доступа для оператора Utel.
iparam gsm.apn 8Настроить точку доступа для оператора Киевстар 3G.
out on или iparam out.1 1Активировать выход.
out off или iparam out.1 0Деактивировать выход.
iparam out.1 8Включить режим контроля датчика движения.
alarm confirmПодтверждение получения сигнала тревоги.
alarm cancelОтмена сигнала тревоги.
alarm startДистанционный запуск тревоги.
